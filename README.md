## Grade Calculator
Calculates average grade from multiple subjects.

**How to use:**
1. Open notebook in Google Colab
2. Run all cells
3. Enter number of subjects
4. Enter marks for each subject
5. Get average and letter grade

**Concepts used:**
- Loops
- Conditionals
- Input/output
